# Dave's AI Clone Engine (Render Deployment)

## Where to put your model files
- Place Wav2Lip files in: `models/wav2lip/checkpoints/`
- Place SadTalker files in: `models/sadtalker/config/`

## How to deploy on Render
1. Upload this entire folder to a new GitHub repo
2. Connect Render to that repo
3. Deploy as a Web Service

Your engine will run at:
 https://daves-ai-clone-engine.onrender.com
